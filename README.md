# infinispan-spark-connector-examples

## Prerequisite: 

### Download and Run Infinispan Cluster

* http://infinispan.org/download/#stable
* Download the latest stable version of Infinispan
* Unzip and cd <ISPN_HOME>
* Run bin/domain.sh
